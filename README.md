# DIGITAL-LITERACY-
Digital Literacy project covering safe internet practices, digital tools, email etiquette, and professional online presence. Includes a cybercrime case study and prevention guide. Built using Google Forms for awareness and HackerRank for coding practice.
# 📘 Digital Literacy Project

## 📌 Overview
This project focuses on understanding and promoting digital literacy among students. It covers essential topics required for safe, responsible, and effective use of digital technology.

---

## 🧠 Topics Covered
- 💻 What is Digital Literacy  
- 🛠️ Useful Digital Tools for Students  
- 🔐 Safe Internet Practices  
- 🌐 Professional Online Presence  
- 📧 Email Etiquette  
- 🚨 Cybercrime Awareness Case Study  
- ✅ Prevention Checklist  

---

## 🛠️ Tools & Platforms Used
- 📊 Google Forms – for Digital Literacy Awareness Quiz  
- 💡 HackerRank – for coding practice  
- 🌐 LinkedIn – for professional networking  
- 💻 GitHub – for project documentation  

---

## 🎯 Objectives
- To spread awareness about digital literacy  
- To promote safe and responsible internet usage  
- To develop technical and communication skills  
- To understand cybercrime risks and prevention
